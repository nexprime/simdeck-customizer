The SimDeck

What is a SimDeck?

A SimDeck is what we call a USB input device consisting of a number of discrete buttons and switches - commonly referred to as a "control panel" or a "button box".

The primary purpose for a SimDeck is to be used as in input device in a simulator environment - flight, space, driving, or otherwise.

The advantages of using a SimDeck include:
  • greatly enhancing the user's immersion in the simulation by providing a set of physical controls which closely resemble those found in the piece of hardware being simulated;
  • greatly increasing the number of controls the user is able to utilize with ease by aiding muscle-memory with varied tactile feel and logical grouping of controls - especially when compared to using a keyboard;
  • greatly simplifying the control scheme in many cases by allowing functions in the simulation to be mapped to a unique button press or a flick of a switch - as opposed using 2-, 3-, and sometimes 4-key shortcuts on the keyboard.

TL;DR
    It's a USB gamepad with 30+ buttons.

Compatibility

Will it work with my OS or software?

The hardware behind each SimDeck is designed to function as any regular USB HID (human interface device) and will appear in the operating system as a "joystick".

Unfortunately it is impractical for us to test every single hardware and software combination. However the rule of thumb to follow is if your hardware and software are able to easily use any and all off-the-shelf USB peripheral (specifically joysticks and gamepads), there should not be any problems with a SimDeck either.

In Windows 7 and later, as well as just about any Linux distro, the drivers are automatically detected as soon as the SimDeck is plugged into the USB port. No additional software of any sort needs to be installed.

Any piece of software which allows custom control bindings from a joystick or gamepad will be able to detect input from the SimDeck just as well.


TL;DR
    Windows, Linux, Mac - good to go!
    Xbox, PlayStation, etc - not so much.
    Software where a joystick can be used (pretty much all PC sim games) - good to go!
    Software where only keyboard and mouse can be used (i.e. Excel) - not so much.      

Construction

How is a SimDeck constructed?

Each SimDeck is hand-assembled from a variety of components.

The outer enclosure is sourced by Hammond Manufacturing - an electrical and electronic hardware manufacturer with over 100 years in the business.
The enclosure is made from 14-gage (1.6mm thick!) powder-coated aluminum and sits on a set of grippy rubber feet.

The buttons and switches are from various sources however, in all cases they have been chosen for their superior tactile feel and aesthetic appearance. The buttons are not mushy and provide definitive feedback when pressed; the flat anti-rotation handles on the toggle switches are easy to actuate in a hurry. See "Types of Inputs" for more about these.

The microprocessor running the device is an ARM Cortex-M0+, a very capable processor with plenty of power in reserve. In fact this is the same processor the researchers at University of Michigan used in 2018 for their "world's smallest computer".

Internally, all electronics are mounted on custom designed and professionally manufactured circuit-boards and wired together with ribbon cable. This provides the greatest degree of electrical stability and resistance to vibration and shock.

Before delivery each SimDeck is thoroughly tested for functionality and finish.


TL;DR
    You could hammer nails with this thing, but please don't - the finish my get scratched :)

Using the SimDeck

How easy is it to use the SimDeck?

Absolutely no software or drivers need to be installed - the SimDeck is completely plug-in-play:
  • Plug in the USB cable
  • Give your computer a minute to recognize the new device
  • You're ready to go!
The rest is entirely upto the software you're using. In the case of my flight-sims, you'd need to go into the options to configure the controls and assign buttons to functions.


TL;DR
    Not any more difficult than setting up a basic keyboard.

Multiple SimDecks

Is it possible to use multiple SimDecks at the same time?

Yes, absolutely!

The only "gotcha", is that certain software might keep a separate controls profile depending on the number of SimDecks plugged in. Meaning it is not an issue if you're not planning on removing and adding SimDecks on a regular basis.

TL;DR
    Yes!

Placing an Order


Currently we accept order requests directly via our email: borealis.simdecks@gmail.com and payment via PayPal.

To place an order, please email us the following:
  • Your shipping address
  • Shipping method you'd like to use (Economy or Express)
  • The quantity of SimDecks you'd like
  • If ordering a custom layout, please remember to include the output from the customization utility
Once we process your request will receive a PayPal invoice, including a shipping quote.

Once the invoice is paid, the assembly of your SimDeck will begin!

Delivery Time

We strive to have each order complete in less than three weeks from the moment invoice is paid.

In vast majority of cases, you will hopefully find the delivery time to be shorter than that.

Since we are not currently able to keep each combination of button and switch in stock in mass quantities we order them on as-needed basis. Therefore the vast majority of the three-week delivery time we expect to be waiting for parts to arrive.

You will also be kept updated, especially If there are delays beyond the three-week time line.
